Sales & Pricing Dashboard
=========================

An interactive dashboard built with Python, Seaborn, Plotly, and Dash.
This project demonstrates statistical visualization, interactivity,
and professional dashboard architecture.

--------------------------------------------------
Project Structure
--------------------------------------------------

dashboard.ipynb      - Exploratory analysis
dashboard.py         - Production Dash app
visualizations/      - Modular plotting & styling
requirements.txt     - Dependencies
dashboard_demo.gif   - Demo preview

--------------------------------------------------
Features
--------------------------------------------------

- Seaborn statistical plots (box, violin, bar, histogram, scatter)
- Interactive Plotly visualizations
- Responsive Dash dashboard layout
- Cohesive color theme and clean UI

--------------------------------------------------
Installation & Run
--------------------------------------------------

pip install -r requirements.txt
python dashboard.py



--------------------------------------------------
Notes
--------------------------------------------------

- Notebook is for exploration only
- Production code lives in dashboard.py
- Styling is centralized to avoid errors
